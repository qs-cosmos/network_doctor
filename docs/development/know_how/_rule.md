# 命令通识

## 参数类型

参考资料

1. 无修饰符: 原生参数

说明文档中的字符即为命令需要使用的字符, 这种参数在使用时必需指定, 且和说明文档里保持一致。

2. <>  : 占位参数

和原生参数类似，都是必需指定的，只不过占位参数的实际字符是在使用时指定的，同时为了方便阅读会用一个描述词汇来表示，并以<>包围。

3. []  : 可选组合

括号里的参数为可选参数, 和占位参数也可以同时使用。

4. ()/{}  : 必选组合

括号里的参数必需指定, 通常里面会是一些互斥参数。

5.  |  : 互斥参数

互斥参数一般都在() 和 [] 里, 表示该参数只能指定其中一个。

6. ... : 可重复指定前一个参数

表示前一个参数可以被指定多个。

7. --  : 标记后续参数类型

表示后续参数的某种类型。
